About Administration Interface Features
  • Dashboard — View the current status of all registered cargo at a glance.
  • Cargo Details — See origin, destination, deadline, and full itinerary for any cargo.
  • Modify Cargo — Change destination or arrival deadline when plans shift.
  • Book Cargo — Register new cargo with origin, destination, and deadline.
  • Route Assignment — Assign optimal itineraries to unrouted cargo.
  • Internal Tracking — Track cargo via the administration tracking interface.
  • Live Map — Monitor all cargo in real time with continuous updates.

Visit eclipse-ee4j.github.io/cargotracker for more information about Cargo Tracker.
